# Visual Prototype Prompt

Use this English prompt with an image generation model to create a product prototype concept image.

```text
Create a polished high-fidelity product prototype board for a Windows desktop companion application called "Desktop Pet Connect".

Show a personalized human-inspired desktop companion avatar floating on a transparent Windows desktop, with a small speech bubble, name tag, and subtle always-on-top presence. The product should feel like a gentle emotional companion layer for friends, partners, and loved ones, not a noisy chat app and not an animal pet product.

Include multiple UI screens in one cohesive prototype composition:
1. The main desktop pet floating near the lower-right corner of a Windows desktop, with a speech bubble showing a friend's message.
2. A compact companion studio window with a fixed left sidebar and focused pages: Home, Make Companion, Messages, Connections, Privacy, Advanced.
3. The Connections page should show: server connection status, user nickname, companion status, invite code creation, copy invite button, add friend by invite code, friend list with online/offline status, and send short message to a friend.
4. A companion package import preview screen showing an avatar package preview image, grid/cell information, validation results, warnings, and an "Import and Use" button.
5. A small friend-companion status card showing a loved one's desktop companion online, with a short message delivered through the companion.

Visual style:
- Modern Windows desktop utility design, dark mode, compact but elegant.
- Refined buttons with clear primary and secondary actions, warm rose and coral accents, sage green online status, subtle shadows, crisp spacing.
- Avatar characters should be customized, expressive, and readable, representing real people through stylized companion forms; avoid animal mascots.
- Avoid marketing hero layouts; this is an actual usable product interface.
- No blue tech style, no beige or brown dominance, no purple-gradient, no decorative orb background. Use a balanced dark neutral palette with rose, coral, soft amber, and sage accents.
- Use clean Chinese UI labels if text appears inside the product screens, but keep the overall generated image description language neutral and product-focused.
- The composition should look like a real software product blueprint/prototype ready for investor or user feedback.

Key product idea:
This app lets people turn a real friend, partner, or loved one into a custom desktop companion, import companion packages, add friends by invite code, see each other's online status, and send real-time short messages through their desktop companions using a WebSocket server.

Aspect ratio: 16:9. High detail. Sharp UI. Professional product prototype. No cartoon-only poster, no landing page, no abstract mascot-only illustration, no animal pet imagery.
```
